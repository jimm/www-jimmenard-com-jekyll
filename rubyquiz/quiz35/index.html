Here is my solution to Ruby Quiz 35: Chess Variants (I). My solution depends upon Bangkok, my chess-game-replaying library.

To use this solution, you need to install Bangkok (which in turn requires midilib) as well as the following files:

  • board.rb
  • chessgame.rb
  • displaylistener.rb
  • main.rb

Alternately, you can download playchess.tar.gz, which contains the directory "playchess" that holds all four files.

% gem install bangkok
...
% copy_these_files_to_your_machine
% cd playchess
% main.rb
[board is drawn]
White: f4
[board is drawn]
Black: Nf6
...
