CheckbooX is a Mac OS X application that mimics the register included with your checkbook. It comes complete with source code. It's the application I used as an example in my article Unit Testing with OCUnit, published on O'Reilly's MacDevCenter on April 23, 2004.

Since I've just released CheckbooX for the first time, there's not much more to say. Please see the file CheckbooX Readme.rtf; it is also included with the application.

Download CheckbooX (, including source).
